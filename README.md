### Paso 1 — Clonar el repositorio
gh repo clone Transportee/podcast

### Paso 3 — Acomodar los archivos
Colocar la carpeta en: `C:/xampp/htdocs/`

### Paso 3 — Crear la base de datos
1. Abrir phpMyAdmin en `http://localhost/phpmyadmin`
2. Crear una base de datos llamada `podcast` con cotejamiento `utf8mb4_unicode_ci`
3. Ir a Importar → seleccionar `database.sql` → Ejecutar

### Paso 4 — Abrir en el navegador
Con XAMPP corriendo, acceder a:
http://localhost/podcast
